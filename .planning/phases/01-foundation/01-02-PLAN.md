   - `setTaxYear(year: number): void`
   - `addEvent(event: Omit<DonationEvent, 'id' | 'items'>): void` â€” generates id via `crypto.randomUUID()`

   - `addItem(eventId: string, item: Omit<DonationItem, 'id'>): void` â€” generates id via `crypto.randomUUID()`
   - `updateItem(eventId: string, itemId: string, patch: Partial<DonationItem>): void`
   - `removeItem(eventId: string, itemId: string): void`

   Persist config:
   - `name: 'donation-itemizer'`
   - `storage: createJSONStorage(() => localStorage)`
   - `version: 1`
   - `partialize`: only persist `schemaVersion`, `taxYear`, `events` (never UI state)
   - `migrate`: placeholder that returns persistedState cast to store type

   Export the store type: `export type DonationStore = { schemaVersion: 1; taxYear: number; events: DonationEvent[]; setTaxYear: ...; addEvent: ...; etc. }`
